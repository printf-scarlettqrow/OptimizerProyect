# OptimizerProyect: Optimizador de Rutas entre Entregas
Este repositorio contiene un proyecto para optimizar rutas de entrega utilizando los datos del dataset llamado "`Order Delivery Dataset`".

## Descripción
Este proyecto consiste en un minisoftware programado en Python creado para calcular el trayecto más corto entre varios destinos de reparto. Para lograrlo primero se limpiará la información y luego se ejecutará el algoritmo correspondiente.

## Dataset
La información proviene de registros reales alojados en Kaggle bajo el enlace:
- **Order Delivery Dataset:** https://www.kaggle.com/datasets/waddahali/order-delivery-dataset

## Requisitos
Para ejecutar este código es necesario tener instalado Python junto con la librería `pandas` para procesar y purificar toda la información
